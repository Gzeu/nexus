### Prerequisites

- Rust 1.75+ ([Install Rust](https://rustup.rs/))
- Git

### Build from Source

```bash
# Clone the repository
git clone https://github.com/Gzeu/nexus.git
cd nexus

# Build the project
cargo build --release

# Install globally
cargo install --path crates/cli
```
